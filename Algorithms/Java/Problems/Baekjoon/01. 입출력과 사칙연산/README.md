#문제
> Hello World!를 출력하시오.

#풀이
~~~java
public class Main {
	public static void main(String[] args) {
        System.out.println("Hello World!");
	}
}
~~~

#문제
> 강한친구 대한육군 을 한 줄에 한 번씩 출력..한다?

#풀이
~~~java
public class Main {
	public static void main(String[] args) {
        System.out.println("강한친구 대한육군");
        System.out.println("강한친구 대한육군");
	}
}
~~~
